# Dotfiles

My desktop Gentoo Linux [dotfiles](http://dotfiles.github.io) powered by [dotbot](https://github.com/anishathalye/dotbot).

## Installation

```
git clone https://github.com/kaiyanlee/dotfiles
cd dotfiles
./install
```

## Optional Dependencies

- linux
- python-wal
- vim
- python
- i3wm
- radare2
- fish
- ranger
- htop
- vscode
